## Deskripsi Proyek

Proyek ini merupakan sebuah media pembelajaran interaktif berbasis web yang membahas tentang teknologi Cloud Computing. Website ini dirancang untuk memenuhi persyaratan tugas mata kuliah dengan mengintegrasikan empat elemen utama media interaktif: teks, gambar, audio, dan animasi.

## Akses Proyek

Seluruh kode sumber dan demonstrasi langsung proyek ini dapat diakses melalui tautan berikut:

- **Live Page**: [https://prayaadikk.github.io/smi-cloud-computing/](https://prayaadikk.github.io/smi-cloud-computing/)
- **Repositori GitHub**: [https://github.com/PrayaaDikk/smi-cloud-computing](https://github.com/PrayaaDikk/smi-cloud-computing)

## Komponen Media

Website ini mencakup berbagai elemen yang saling berinteraksi untuk memberikan pengalaman belajar yang komprehensif:

1.  **Teks**: Penjelasan mendalam mengenai pengertian, sejarah, pilar layanan (IaaS, PaaS, SaaS), serta model deployment cloud.
2.  **Gambar**: Ilustrasi visual dan penggunaan komponen gambar yang memperjelas konsep-konsep teknis yang disampaikan.
3.  **Audio**: Fitur audio widget yang memungkinkan pengguna memutar musik latar untuk meningkatkan fokus saat belajar melalui kontrol interaktif.
4.  **Animasi**: Penggunaan animasi canvas untuk latar belakang grid, transisi scroll reveal, serta animasi pada sistem kuis interaktif.

## Fitur Utama

- **Navigasi Interaktif**: Menu navigasi yang memudahkan pengguna berpindah antar bagian materi secara halus.
- **Sistem Kuis**: Evaluasi pemahaman pengguna di akhir materi dengan umpan balik skor secara instan.
- **Visual Modern**: Desain estetik menggunakan tipografi Barlow dan DM Serif Display dengan palet warna yang terstruktur.
- **Responsivitas**: Tata letak yang bersih dengan pengorganisasian elemen yang sistematis.

## Instruksi Penggunaan

Untuk mendapatkan pengalaman visual dan interaktivitas terbaik, pengguna sangat disarankan untuk mengakses website ini melalui perangkat **Desktop** atau Laptop. Beberapa elemen animasi dan tata letak dirancang khusus untuk layar lebar guna memastikan semua informasi tersampaikan dengan jelas dan fungsionalitas interaktif berjalan optimal.
